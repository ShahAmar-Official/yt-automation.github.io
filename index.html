How It Works

1

🔍 Find Trending Topics

Discovers the hottest topics from Google Trends and Reddit in real time.

pytrends + Reddit API (free)
2

✍️ Generate Script

Creates an engaging narration with title, scenes, tags, and description using smart templates.

Template engine (free, no API)
3

🎙️ Text-to-Speech

Converts the script into natural-sounding speech audio.

gTTS — Google Text-to-Speech (free)
4

🎬 Create Video

Assembles portrait video with stock footage, captions, and audio.

MoviePy + Pexels API (free)
5

🖼️ Generate Thumbnail

Designs an eye-catching 1280 × 720 thumbnail with dynamic text.

Pillow (free)
6

🚀 Upload to YouTube

Publishes the video with optimised metadata and thumbnail.

YouTube Data API v3 (free)

Quick Setup

  1. Fork this repository — click the Fork button on GitHub
  2. Set up YouTube OAuth2 credentials — via Google Cloud Console (free)
  3. Get a Pexels API key — sign up at pexels.com/api (free)
  4. Add secrets to GitHub — YOUTUBE_CLIENT_SECRET, YOUTUBE_TOKEN, PEXELS_API_KEY
  5. Enable GitHub Actions — the pipeline runs automatically every 6 hours
